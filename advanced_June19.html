Software Developers CoP

Building Complex Web Applications June 2019

The class took place June 3rd - June 6th 2019 at the RMC Denver office. There were 12 students and our fearless teacher Will B.

The course covered the following topics:

  • Usin Node.js and NPM for development
  • How to use Redux bundler
  • Building user interfaces using components in React
  • Using JWT and Corpsmap packages to authenticate users via CAC cards
  • Posting and retrieving data from a no-sql database
